1. Tag b: It is used to make a text bold.

  2. Tag i: It is used to represent a text in some different voice.

  3. Tag u: It is used to render enclosed text with an underline.

  4. Tag p: It represents a paragraph in an HTML document.

  5. Tag br: br stands for break line, it breaks the line of the code.

  6. Tag hr: hr stands for Horizontal Rule. This tag is used to put a line across the webpage.

  7. Tag aside: It defines content aside from main content. Mainly represented as sidebar.

  8. Tag button: It is used to represent a clickable button

  9. Tag canvas: It is used to provide a graphics space within a web document.

  10. Tag div: It defines a division or section within HTML document.
